## 📖 Features / Возможности

| 🇬🇧 English | 🇷🇺 Русский |

## 🚀 How to use / Как использовать

### 🔧 Local setup / Локальный запуск
1. Save the `player.html` file anywhere.  
   Сохраните файл `player.html` в любом месте.
2. Double‑click to open in any modern browser (Chrome, Firefox, Edge, etc.).  
   Дважды кликните, чтобы открыть в любом современном браузере (Chrome, Firefox, Edge и т.д.).
3. No internet connection required after the page is loaded – works fully offline.  
   После загрузки страницы интернет не нужен – плеер работает полностью офлайн.

### 🎛️ Basic workflow / Основная работа

1. **Add tracks** – either paste a direct MP3 URL or pick a local audio file.  
   **Добавьте треки** – вставьте прямую ссылку на MP3 или выберите локальный файл.
2. **Manage playlist** – play, remove, save/load playlist as JSON.  
   **Управляйте плейлистом** – воспроизводите, удаляйте, сохраняйте/загружайте плейлист в JSON.
3. **Adjust sound** – use the 10‑band EQ, bass/treble, volume, reverb.  
   **Настройте звук** – используйте 10-полосный эквалайзер, бас/ВЧ, громкость, реверберацию.
4. **Switch visualization** – click "2D/3D" to toggle.  
   **Переключайте визуализацию** – нажмите «2D/3D» для смены режима.
5. **Save your settings** – export EQ presets or effect presets to reuse later.  
   **Сохраните настройки** – экспортируйте пресеты эквалайзера или эффектов для повторного использования.
